# 《囚城营救》核心剧情设定设计

## 项目定位

《囚城营救》是基于《九州缥缈录》原著小说人物与世界观创作的原创同人支线。

原著人物、势力与人物关系归原作者及相关权利方所有。本项目原创内容仅包括本次潜入营救支线、关卡流程与玩法实现，不声称原作设定原创。

## 核心剧情

- 主角：姬野。
- 营救目标：吕归尘。
- 敌方势力：离国军队。
- 事件背景：吕归尘在战争中被离国军俘获，被关押于离国边境城堡接受审问。
- 营救动机：姬野为了营救昔日战友吕归尘，独自潜入城堡。
- 限时原因：离国军将在黎明前把吕归尘转移至主力军营。
- 游戏目标：姬野必须在押送队出发前潜入城堡并找到吕归尘。
- 通关条件：姬野找到吕归尘。
- 失败条件：倒计时结束前未找到吕归尘。

## 内容边界

本阶段不加入：

- 找到吕归尘后的护送或逃离玩法。
- 离国军审问的具体情报内容。
- 原著中未确认的人物经历或关系。
- 新的势力、角色或结局。

## RAG 验收问题

1. `姬野为什么要潜入城堡？`
   - 应回答为了营救昔日战友吕归尘。
2. `吕归尘被谁关押在哪里？`
   - 应回答被离国军关押于离国边境城堡。
3. `为什么营救任务有时间限制？`
   - 应回答离国军将在黎明前转移吕归尘。
4. `怎样才算完成关卡？`
   - 应回答姬野找到吕归尘即完成。
5. `找到吕归尘后是否需要护送他逃离？`
   - 当前知识库没有该玩法，应拒绝编造或明确当前版本不包含护送逃离。
